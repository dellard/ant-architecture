Ant-8 Documentation - aa8

Name

aa8 -- The Ant-8 assembler.

Synopsis

aa8 [options] filename

Description

aa8 assembles the given file (which must be an Ant-8 assembly language program), creating an Ant-8 machine language program that can be run on the Ant-8 processor.

The name of the output file is determined by the name of the input file: if the input filename ends in .asm, then the .asm is replaced with .ant. If the filename does not end in .asm, then the suffix .ant is simply appended to the filename.

If the input file does not contain a valid Ant-8 assembly language program, then aa8 attempts to print useful diagnostic messages to tell the user something about what is wrong with the program.

Options

-h

Print a usage message and then exit.

-V

Print the program name and version, and then exit.
