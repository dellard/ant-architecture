Ant-8 Documentation - ant8

Name

ant8 - The Ant-8 virtual machine.

Synopsis

ant8 [options] filename

Description

ant8 executes the given file (which must be an Ant-8 machine language program). This is accomplished by running the program on a virtual Ant-8 processor (implemented in C).

Options

-h

Print a usage message and then exit.

-V

Print the program name and version, and then exit.

-d

After executing the program, dump the contents of Ant-8 memory to file ant8.core.
